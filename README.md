# Snooker Masters

A professional web-based snooker and 8-ball pool game built with HTML, CSS, and JavaScript.

## About

Snooker Masters is an interactive browser game that brings the excitement of cue sports to your desktop. The game features two popular variants:

- **Snooker**: The classic game with 15 red balls, 6 colored balls, and precise scoring mechanics
- **8-Ball Pool**: The American pool variant where players must pocket their designated balls and the 8-ball to win

## Features

- Realistic physics simulation for ball movement and collisions
- Intuitive mouse controls for aiming and shooting
- Score tracking and game state management
- Multiple game modes (Snooker and 8-Ball)
- Responsive design that works on different screen sizes
- Sound effects for an immersive experience

## How to Play

### Snooker Rules
- Pot the 15 red balls first, then the colored balls in order
- Each ball has different point values
- The player with the highest score wins

### 8-Ball Rules
- Players are assigned stripes or solids
- Pocket all your balls, then the 8-ball to win
- Avoid pocketing the 8-ball before clearing your set

## Getting Started

1. Clone the repository
2. Open `index.html` in your web browser
3. Choose your game mode and start playing!

## Technologies Used

- HTML5
